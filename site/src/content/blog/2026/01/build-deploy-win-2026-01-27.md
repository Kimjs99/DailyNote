오늘은 마침내 정적 블로그를 완성하고 GitHub Pages에 배포까지 마쳤다. 지금
시각은 2026-01-27 17:02. 처음 요구사항을 정리하던 단계부터 디자인 톤을
조정하고, 글래스모피즘을 최대치로 끌어올리는 과정까지 이어졌다. 결국
모든 페이지가 정상 작동하고, 홈과 글 목록, 태그/카테고리 페이지까지 문제
없이 연결된 걸 확인했을 때 확실한 성취감이 들었다.

## 오늘의 진행 요약

- Astro 기반 정적 블로그 구조 확정
- 한글 중심 UI 문구와 날짜 포맷 적용
- 글래스모피즘 강화 및 포인트 컬러 정리
- 대표 아이콘과 파비콘 연결
- GitHub Pages 배포 워크플로 정비

## 배포 과정에서 기억할 점

1. `base` 경로가 붙는 GitHub Pages 환경에서는 내부 링크를 반드시
   `BASE_URL` 기반으로 작성해야 했다.
2. Actions에서는 `pnpm-lock.yaml` 경로가 `site/` 아래에 있어
   캐시 경로 설정이 필요했다.
3. 배포 확인 후 404가 계속 나오면, 라우팅과 링크 경로부터 재점검하는 게
   가장 빠른 해결책이었다.

## 오늘의 느낀 점

작은 수정들이 쌓여 최종 결과물이 되었고, 배포가 성공했을 때
"드디어 내 블로그가 온라인에 있다"는 확신이 들었다. 앞으로는
이 공간에 개발 과정과 배운 내용을 더 자주 기록해 나가고 싶다.
