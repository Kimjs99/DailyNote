# 🏃‍♂️ 체육수업 소개 웹사이트

## 📋 프로젝트 개요

체육수업을 소개하는 반응형 원페이지 웹사이트입니다. 현대적이고 깔끔한 디자인으로 체육수업의 특징, 시간표, 강사 정보, 문의사항을 한눈에 볼 수 있습니다.

## 🎯 주요 기능

- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기에서 최적화된 화면
- **원페이지 스크롤**: 부드러운 스크롤 애니메이션으로 사용자 경험 향상
- **현대적 UI/UX**: 그라데이션 배경과 카드형 레이아웃으로 시각적 매력도 증대
- **인터랙티브 요소**: 호버 효과와 스크롤 시 헤더 변화 효과

## 🏗️ 기술 스택

- **HTML5**: 시맨틱 마크업 구조
- **CSS3**: 
  - Flexbox & Grid 레이아웃
  - CSS 그라데이션
  - CSS 애니메이션 및 트랜지션
  - 미디어 쿼리를 통한 반응형 디자인
- **JavaScript**: 
  - 부드러운 스크롤 효과
  - 스크롤 이벤트 처리

## 📱 섹션 구성

### 1. 헤더 (Header)
- 로고 및 네비게이션 메뉴
- 스크롤 시 배경 투명도 변화
- 반응형 모바일 메뉴

### 2. 히어로 섹션 (Hero Section)
- 메인 타이틀 및 부제목
- CTA(Call-to-Action) 버튼
- 체육시설 배경 이미지 (50% 투명도)
- 그라데이션 오버레이

### 3. 수업 소개 (About Section)
- **체력 향상**: 근력, 지구력, 유연성 종합 향상
- **맞춤형 수업**: 개인별 체력 수준에 맞는 프로그램
- **소그룹 수업**: 개별 맞춤 지도와 동기부여

### 4. 시간표 (Schedule Section)
- 월/수/금/토 수업 일정
- 카드형 레이아웃으로 깔끔한 정보 표시
- 각 수업별 상세 내용 포함

### 5. 강사 소개 (Instructor Section)
- 김체육 선생님 프로필
- 전문 자격 및 경력 정보
- 그라데이션 배경으로 강조

### 6. 문의 정보 (Contact Section)
- 연락처, 이메일, 위치, 운영시간
- 다크 테마로 시각적 구분
- 반응형 그리드 레이아웃

## 🎨 디자인 특징

### 색상 팔레트
- **Primary**: `#667eea` (보라-파랑)
- **Secondary**: `#764ba2` (보라)
- **Accent**: `#ff6b6b` (코랄)
- **Background**: 그라데이션 배경

### 타이포그래피
- **Font Family**: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif
- **계층적 텍스트 크기**: 3.5rem (메인 타이틀) ~ 0.9rem (부가 정보)

### 레이아웃
- **Container**: 최대 1200px 너비, 중앙 정렬
- **Grid System**: CSS Grid를 활용한 반응형 레이아웃
- **Card Design**: 그림자와 둥근 모서리로 현대적 느낌

## 🚀 사용 방법

1. **파일 다운로드**: `test.html` 파일을 다운로드
2. **브라우저에서 열기**: 웹 브라우저에서 HTML 파일 열기
3. **반응형 테스트**: 다양한 화면 크기에서 테스트

## 📱 반응형 브레이크포인트

```css
/* 모바일 */
@media (max-width: 768px) {
    .nav-links { display: none; }
    .hero h1 { font-size: 2.5rem; }
    .hero p { font-size: 1.1rem; }
    .section h2 { font-size: 2rem; }
}
```

## 🔧 커스터마이징

### 색상 변경
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #ff6b6b;
}
```

### 배경 이미지 변경
```css
.hero {
    background-image: url('your-image-url.jpg');
}
```

### 텍스트 내용 수정
HTML 파일에서 각 섹션의 텍스트 내용을 직접 수정할 수 있습니다.
