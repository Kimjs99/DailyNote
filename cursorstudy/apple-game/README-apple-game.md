# 🍎 사과 게임 (Apple Game)

HTML, CSS, JavaScript로 만든 재미있는 숫자 조합 퍼즐 게임입니다!

## 🎮 게임 소개

사과 게임은 1-9 숫자가 적힌 사과들을 마우스로 드래그하여 선택하고, 선택한 사과들의 숫자 합이 10이 되면 사과들이 사라지는 퍼즐 게임입니다. 1분 안에 최대한 많은 사과를 제거하여 높은 점수를 획득하는 것이 목표입니다.

## ✨ 주요 기능

### 🎯 게임 플레이
- **15x10 그리드**: 총 150개의 사과가 배치된 게임 보드
- **드래그 선택**: 마우스로 드래그하여 여러 사과를 한 번에 선택
- **숫자 조합**: 선택한 사과들의 숫자 합이 10이 되면 사과들이 사라짐
- **점수 시스템**: 제거한 사과 개수만큼 점수 획득

### ⏰ 시간 제한
- **1분 타이머**: 60초 제한 시간으로 긴장감 있는 게임
- **실시간 표시**: 남은 시간을 실시간으로 확인 가능
- **경고 표시**: 10초 이하일 때 빨간색으로 경고

### ⏸️ 일시정지 기능
- **멈춤/계속**: 게임 중간에 일시정지 가능
- **사과 숨김**: 멈춤 시 모든 사과가 숨겨져 깔끔한 화면
- **상태 표시**: 버튼 색상과 텍스트로 현재 상태 확인

### 🎨 시각적 효과
- **아름다운 디자인**: 그라데이션과 그림자 효과
- **애니메이션**: 사과 제거 시 축하 애니메이션
- **반응형 디자인**: 다양한 화면 크기에 최적화

## 🚀 게임 실행

1. `apple-game.html` 파일을 웹 브라우저에서 열기
2. 게임이 자동으로 시작됩니다
3. 마우스로 사과들을 드래그하여 선택하세요!

## 🎯 게임 방법

### 기본 조작
1. **사과 선택**: 마우스를 드래그하여 원하는 사과들을 선택
2. **조합 확인**: 선택한 사과들의 숫자 합이 10인지 확인
3. **사과 제거**: 합이 10이 되면 자동으로 사과들이 사라짐
4. **점수 획득**: 제거한 사과 개수만큼 점수 획득

### 전략 팁
- **빠른 조합**: 1+9, 2+8, 3+7, 4+6, 5+5 등의 조합을 찾아보세요
- **여러 사과**: 1+2+7, 1+3+6 등의 3개 이상 조합도 가능합니다
- **시간 관리**: 1분 안에 최대한 많은 사과를 제거하세요

## 🎯 게임 특징

### 고유한 게임 메커니즘
- **빈 공간 유지**: 사과가 제거되면 그 자리가 비어있는 상태로 유지
- **실시간 합계**: 선택한 사과들의 합계를 실시간으로 표시
- **시각적 피드백**: 선택, 제거, 축하 등 다양한 시각적 효과

### 사용자 경험
- **직관적 조작**: 마우스 드래그만으로 간단한 조작
- **즉시 피드백**: 선택과 제거에 대한 즉각적인 시각적 반응
- **명확한 상태 표시**: 점수, 시간, 선택 합계 등 모든 정보를 명확히 표시

## 🏆 점수 시스템

- **기본 점수**: 제거한 사과 1개당 1점
- **시간 제한**: 1분 안에 최대한 많은 사과 제거
- **최종 점수**: 게임 종료 시 총 제거한 사과 개수 표시

## 🔧 커스터마이징

게임을 수정하고 싶다면:

1. **사과 개수**: `initGame()` 함수에서 `totalApples` 값 변경
2. **그리드 크기**: CSS의 `grid-template-columns` 값 수정
3. **게임 시간**: `timeLeft` 초기값 변경
4. **사과 크기**: CSS의 `.apple` 클래스 크기 값 수정
