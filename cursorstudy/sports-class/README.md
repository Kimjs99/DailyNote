# 🏃‍♂️ Sports Class

HTML, CSS, JavaScript를 사용한 체육 수업 관련 웹사이트입니다.

## 📁 파일 구조
- `index.html` - 메인 HTML 파일
- `style.css` - 스타일시트
- `script.js` - JavaScript 기능
- `P.png` - 프로젝트 이미지
- `자기소개서 실습용.txt` - 실습용 텍스트 파일
- `README.md` - 프로젝트 설명서

## 🎯 프로젝트 목적
체육 수업과 관련된 웹사이트 개발 실습

## 🚀 실행 방법
브라우저에서 `index.html` 파일을 열면 웹사이트를 확인할 수 있습니다.
