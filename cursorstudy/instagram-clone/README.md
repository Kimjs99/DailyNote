# 📸 Instagram Clone

React를 사용하여 만든 인스타그램 스타일의 SNS 앱입니다.

## 📁 파일 구조
- `instagram-app.html` - 메인 앱 파일 (React 코드 포함)
- `instagram-style.css` - 인스타그램 스타일의 CSS

## ✨ 주요 기능
- 📱 인스타그램과 유사한 UI/UX
- 📸 피드 포스트 표시 (좋아요, 댓글 기능)
- 📖 스토리 섹션
- 💖 인터랙티브 좋아요 버튼
- 💬 댓글 보기/숨기기 기능

## 🎨 디자인 특징
- 따뜻한 모노톤 색상 팔레트
- 반응형 디자인 (모바일/데스크톱 지원)
- 인스타그램 브랜드 컬러 그라데이션

## 🚀 실행 방법
브라우저에서 `instagram-app.html` 파일을 열면 앱을 사용할 수 있습니다.

## 🖼️ 사용된 이미지
- 프로필 사진: `shared-assets/images/character-images/` 폴더의 man.png, woman.png
- 피드 이미지: `shared-assets/images/Images/` 폴더의 img01-10 파일들
