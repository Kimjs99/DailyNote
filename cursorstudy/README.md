# Cursor Study Projects

다양한 프로그래밍 프로젝트들을 모아놓은 저장소입니다. 웹 개발, 데이터 크롤링, 게임 개발 등 다양한 기술 스택을 학습하고 실습한 프로젝트들을 포함합니다.

## 📊 데이터 크롤링 프로젝트

### 1. [금 시세 크롤러](./gold-price-crawler/)
- **설명**: 한국금거래소에서 금 시세 데이터를 수집하고 분석하는 종합 도구
- **기술**: Python, Selenium, Pandas, Matplotlib, Seaborn
- **기능**: 
  - 실시간 금 시세 데이터 크롤링 (순금, 18K, 14K)
  - 상세한 통계 분석 (평균, 변동률, 상관관계)
  - 7가지 차트로 데이터 시각화
  - 엑셀 파일로 데이터 저장 및 정리

### 2. [Yahoo Finance 주식 크롤러](./yahoo-stocks-crawler/)
- **설명**: Yahoo Finance에서 주식 상승률 데이터를 수집하고 분석
- **기술**: Python, Requests, BeautifulSoup, Pandas, Matplotlib
- **기능**:
  - 주식 상승률 데이터 크롤링
  - 무작위 클래스명 처리로 안정적인 크롤링
  - 통계 분석 및 시각화 (7가지 차트)
  - 엑셀 파일로 데이터 저장

## 🎮 게임 프로젝트

### 3. [사과 게임](./apple-game/)
- **설명**: HTML5 Canvas를 사용한 간단한 사과 수집 게임
- **기술**: HTML5, CSS3, JavaScript
- **기능**: 키보드 조작, 점수 시스템, 게임 오버 처리

### 4. [스포츠 클래스 웹사이트](./sports-class/)
- **설명**: 체육 수업 관련 웹사이트
- **기술**: HTML5, CSS3, JavaScript
- **기능**: 반응형 디자인, 인터랙티브 요소

## 📱 웹 애플리케이션

### 5. [메모 앱 (Next.js)](./my-memo/)
- **설명**: Next.js 15와 Express.js를 사용한 현대적인 메모 관리 애플리케이션
- **기술**: Next.js 15, TypeScript, Tailwind CSS, Express.js, Prisma, SQLite
- **기능**: 
  - 사용자 인증 (JWT)
  - 메모 CRUD, 태그 시스템, 색상 구분
  - 검색 기능, 반응형 디자인
  - 실시간 편집 및 미리보기

### 6. [인스타그램 클론](./instagram-clone/)
- **설명**: 인스타그램 UI를 모방한 웹 애플리케이션
- **기술**: HTML5, CSS3, JavaScript
- **기능**: 피드 레이아웃, 반응형 디자인, 인터랙티브 요소

### 7. [체육 수업 연구소](./class-project/my-awesome-peclass/)
- **설명**: Next.js를 사용한 체육 교육 종합 웹 애플리케이션
- **기술**: Next.js 15, TypeScript, Tailwind CSS, Radix UI
- **기능**: 
  - 수업 스케줄 관리 (캘린더)
  - 프로그램 둘러보기
  - 반응형 디자인

## 🛠️ 유틸리티 도구

### 8. [PDF 도구](./pdf-editor/)
- **설명**: PDF 페이지 추출 및 파일 병합을 지원하는 통합 GUI 도구
- **기술**: Python, Tkinter, PyPDF
- **기능**:
  - PDF 페이지 추출 (범위 지정)
  - 다중 PDF 파일 병합
  - GUI 인터페이스
  - macOS 앱으로 패키징

### 9. [QR 코드 생성기](./QR-generator/)
- **설명**: 로고가 포함된 QR 코드를 생성하는 웹 애플리케이션
- **기술**: React 18, TypeScript, Vite, Tailwind CSS
- **기능**:
  - 로고 업로드 및 QR 코드 생성
  - 최대 10개 URL 관리
  - 자동 다운로드
  - 반응형 디자인

## 🛠️ 기술 스택

### Frontend
- **HTML5/CSS3**: 웹 표준 기술
- **JavaScript**: 클라이언트 사이드 프로그래밍
- **React 18**: 컴포넌트 기반 UI 라이브러리
- **Next.js 15**: React 기반 풀스택 프레임워크
- **TypeScript**: 정적 타입 검사
- **Tailwind CSS**: 유틸리티 퍼스트 CSS 프레임워크
- **Radix UI**: 접근성 중심 UI 컴포넌트

### Backend
- **Node.js**: JavaScript 런타임
- **Express.js**: 웹 프레임워크
- **Prisma**: 데이터베이스 ORM
- **SQLite**: 경량 데이터베이스
- **JWT**: 인증 토큰

### 데이터 처리
- **Python**: 데이터 분석 및 크롤링
- **Pandas**: 데이터 조작 및 분석
- **Selenium**: 웹 자동화
- **BeautifulSoup**: HTML 파싱
- **Matplotlib/Seaborn**: 데이터 시각화
- **Requests**: HTTP 클라이언트

### 유틸리티
- **PyPDF**: PDF 처리
- **Tkinter**: GUI 프레임워크
- **QRCode.js**: QR 코드 생성

## 🚀 빠른 시작

### 웹 애플리케이션 실행
```bash
# 메모 앱 (Next.js)
cd my-memo/frontend
npm install
npm run dev

# 체육 수업 연구소
cd class-project/my-awesome-peclass
npm install
npm run dev

# QR 코드 생성기
cd QR-generator
npm install
npm run dev
```

### 데이터 크롤링 실행
```bash
# 금 시세 크롤러
cd gold-price-crawler
pip install -r requirements.txt
python3 gold_crawler.py

# 주식 크롤러
cd yahoo-stocks-crawler
pip install -r requirements.txt
python3 yahoo_stocks_simple.py
```

### 유틸리티 도구 실행
```bash
# PDF 도구
cd pdf-editor
pip install -r requirements.txt
python3 pdf_tool_gui.py
```
