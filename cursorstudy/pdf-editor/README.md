# PDF 도구 - 추출 및 병합

PDF 파일의 페이지 추출과 파일 병합을 모두 지원하는 통합 GUI 도구입니다.

## 🚀 주요 기능

### 📄 페이지 추출
- **GUI 인터페이스**: 사용자 친화적인 그래픽 인터페이스
- **페이지 범위 추출**: 원하는 시작 페이지와 끝 페이지를 지정하여 추출
- **자동 파일명 생성**: 날짜와 시간이 포함된 파일명 자동 생성

### 🔗 파일 병합
- **다중 파일 선택**: 여러 PDF 파일을 한 번에 선택
- **순서 관리**: 파일 순서를 위/아래로 이동하여 병합 순서 조정
- **직관적 인터페이스**: 드래그 앤 드롭으로 파일 순서 변경

### 🛡️ 안전 기능
- **안전한 종료**: 종료 버튼을 통한 안전한 프로그램 종료
- **중복 실행 방지**: 동일한 프로그램이 이미 실행 중이면 새로 실행하지 않음
- **실시간 상태 표시**: 현재 작업 상태를 실시간으로 확인
- **작업 중 종료 방지**: 작업 중일 때는 종료 확인 메시지

## 📁 프로젝트 구조

```
pdf-editor/
├── pdf_tool_gui.py              # 통합 GUI 프로그램 (메인) ⭐
├── pdf_page_extractor.py        # 터미널 버전 (백업용)
├── pdf_extractor_gui_safe.py    # 이전 GUI 버전 (백업용)
├── create_pdf_tool_app.py       # 통합 앱 생성 스크립트 ⭐
├── create_safe_app.py           # 이전 앱 생성 스크립트 (백업용)
├── app_icon.icns                # 앱 아이콘
├── requirements.txt             # 필요한 라이브러리 목록
└── README.md                    # 이 파일
```

## 🛠️ 설치 및 실행

### 1. 라이브러리 설치
```bash
pip install -r requirements.txt
```

### 2. GUI 버전 실행 (권장)
```bash
python3 pdf_tool_gui.py
```

### 3. 앱으로 실행
```bash
python3 create_pdf_tool_app.py
```
생성된 `PDF 도구.app`을 더블클릭하여 실행

### 4. 터미널 버전 실행 (백업용)
```bash
python3 pdf_page_extractor.py
```

## 💻 사용 방법

### 📄 페이지 추출

1. **"📄 페이지 추출" 탭** 선택
2. **PDF 파일 선택**: "찾아보기" 버튼으로 PDF 파일 선택
3. **페이지 범위 설정**: 시작 페이지와 끝 페이지 입력
4. **저장 위치 선택**: 자동 파일명 생성 또는 직접 지정
5. **추출 실행**: "📄 페이지 추출" 버튼 클릭

### 🔗 파일 병합

1. **"🔗 파일 병합" 탭** 선택
2. **파일 추가**: "📁 파일 추가" 버튼으로 병합할 PDF 파일들 선택
3. **순서 조정**: 
   - "⬆️ 위로 이동" / "⬇️ 아래로 이동" 버튼으로 순서 변경
   - "🗑️ 선택 삭제"로 불필요한 파일 제거
4. **저장 위치 선택**: "찾아보기" 버튼으로 병합된 파일 저장 위치 선택
5. **병합 실행**: "🔗 파일 병합" 버튼 클릭

### 🚪 안전한 종료
- **"🚪 안전하게 종료"** 버튼 클릭
- 작업 중일 때는 **종료 확인 메시지**가 나타남
- **"예"**를 선택하면 안전하게 종료

## 📋 출력 파일명 형식

### 페이지 추출
```
원본파일명_pages_시작페이지-끝페이지_YYYYMMDD_HHMMSS.pdf
```

### 파일 병합
사용자가 직접 지정한 파일명으로 저장

**예시:**
- `document_pages_1-5_20241201_143022.pdf` (추출)
- `merged_documents.pdf` (병합)

## 🔧 문제 해결

### 앱이 실행되지 않을 때
- Finder에서 앱을 우클릭 → "열기" 선택
- 터미널에서: `open '/Users/kimpro/Desktop/PDF 도구.app'`

### 중복 실행 오류
- 기존 창을 찾아서 사용하거나
- 터미널에서: `rm /tmp/pdf_tool_gui.lock`

### 라이브러리 오류
```bash
pip install pypdf
```
