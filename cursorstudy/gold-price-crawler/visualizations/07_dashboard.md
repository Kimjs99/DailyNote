# 07_dashboard.png - 통합 대시보드

## 📊 대시보드 개요
6개의 차트와 주요 통계 요약을 한 화면에 통합한 종합 대시보드입니다.

## 🎯 구성 요소

### 1. 순금 가격 추이 (상단 좌측)
- **내용**: 순금 구매가와 판매가의 시간별 변화
- **목적**: 순금 시장의 전체적인 트렌드 파악
- **분석**: 구매가와 판매가 간의 스프레드 확인

### 2. 18K/14K 가격 추이 (상단 중앙)
- **내용**: 18K와 14K 금의 시간별 가격 변화
- **목적**: 합금 금 시장의 동향 파악
- **분석**: 순금과의 가격 연관성 확인

### 3. 가격 상관관계 (상단 우측)
- **내용**: 4가지 금 종류 간의 상관계수 히트맵
- **목적**: 금 종류 간의 연관성 파악
- **분석**: 투자 다각화 가능성 평가

### 4. 순금 가격 분포 (중간 좌측)
- **내용**: 순금 구매가와 판매가의 분포 히스토그램
- **목적**: 가격 분포의 특성 파악
- **분석**: 가격의 안정성과 극값 존재 여부

### 5. 일일 변동률 (중간 중앙)
- **내용**: 순금 구매가의 일일 변동률 분포
- **목적**: 가격 변동의 안정성 평가
- **분석**: 리스크 수준과 변동성 파악

### 6. 기간별 비교 (중간 우측)
- **내용**: 전체기간, 최근30일, 최근7일 평균 가격 비교
- **목적**: 시장 트렌드 변화 파악
- **분석**: 단기 vs 장기 가격 동향

### 7. 통계 요약 (하단)
- **내용**: 주요 통계 수치와 분석 기간 정보
- **목적**: 핵심 정보를 한눈에 파악
- **분석**: 데이터의 신뢰성과 대표성 확인

## 🔍 종합 분석 포인트

### 1. 시장 트렌드
- **전체적인 방향성**: 상승, 하락, 횡보 중 어느 쪽인지
- **변동성 수준**: 안정적, 중간, 불안정 중 어느 수준인지
- **가격 범위**: 현재 가격이 과거 대비 어떤 위치인지

### 2. 투자 관점
- **매수 타이밍**: 현재가가 평균 대비 낮은지
- **매도 타이밍**: 현재가가 평균 대비 높은지
- **리스크 수준**: 변동성이 투자 목표에 적합한지

### 3. 포트폴리오 구성
- **다각화 효과**: 금 종류 간 상관관계가 낮은지
- **비중 조절**: 어떤 금 종류에 더 집중할지
- **리스크 분산**: 변동성이 높은 종류의 비중 조절

## 💡 활용 가이드

### 투자자 관점
1. **현재 시장 상황 파악**: 대시보드 전체를 통해 시장 동향 확인
2. **투자 타이밍 판단**: 가격 추이와 기간별 비교로 매수/매도 시점 결정
3. **리스크 평가**: 변동률과 분포 차트로 리스크 수준 파악
4. **포트폴리오 구성**: 상관관계 차트로 다각화 전략 수립

### 분석가 관점
1. **데이터 품질 확인**: 통계 요약으로 데이터의 신뢰성 평가
2. **패턴 분석**: 각 차트에서 발견되는 패턴과 트렌드 분석
3. **예측 모델링**: 과거 데이터 기반 미래 가격 예측
4. **리스크 모델링**: 변동성과 상관관계 기반 리스크 모델 구축

### 일반 사용자 관점
1. **현재 가격 수준**: 평균 대비 현재 가격이 높은지/낮은지 확인
2. **가격 안정성**: 가격이 얼마나 안정적인지 파악
3. **투자 적합성**: 자신의 투자 목표에 적합한지 판단
4. **시장 이해**: 금 시장의 특성과 동향 이해

## 🎯 대시보드의 장점

### 1. 종합성
- **한눈에 파악**: 여러 차원의 정보를 동시에 확인
- **상호 연관성**: 각 차트 간의 연관성 파악
- **전체적 관점**: 부분이 아닌 전체적인 시장 상황 파악

### 2. 효율성
- **시간 절약**: 여러 차트를 따로 보지 않고 한 번에 확인
- **비교 용이**: 서로 다른 지표들을 쉽게 비교
- **의사결정 지원**: 빠른 의사결정에 필요한 정보 제공

### 3. 직관성
- **시각적 표현**: 숫자보다 차트로 이해하기 쉬움
- **색상 구분**: 각 요소를 색상으로 구분하여 인식 용이
- **구조화**: 논리적인 순서로 정보 배치
