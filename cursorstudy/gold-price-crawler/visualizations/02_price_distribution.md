# 02_price_distribution.png - 금 가격 분포 히스토그램

## 📊 차트 개요
각 금 종류별 가격 분포를 히스토그램과 KDE(커널 밀도 추정) 곡선으로 표현한 이미지입니다.

## 📈 포함된 데이터
- **순금 구매가 분포**: 구매가의 빈도 분포
- **순금 판매가 분포**: 판매가의 빈도 분포
- **18K 판매가 분포**: 18K 금 판매가의 빈도 분포
- **14K 판매가 분포**: 14K 금 판매가의 빈도 분포

## 🔍 분석 포인트

### 1. 분포의 형태
- **정규분포**: 종 모양의 대칭적 분포
- **왜도(Skewness)**: 분포의 비대칭성
- **첨도(Kurtosis)**: 분포의 뾰족함 정도

### 2. 중심 경향성
- **평균값 (빨간 점선)**: 모든 값의 산술 평균
- **중앙값 (초록 점선)**: 중간값, 50% 사분위수
- 평균과 중앙값의 차이로 분포의 비대칭성 파악

### 3. 분포의 집중도
- 히스토그램의 높이로 빈도 확인
- KDE 곡선의 뾰족함으로 집중도 파악

### 4. 이상치(Outlier) 존재
- 분포의 양 끝단에 위치한 극값들
- 정상 범위를 벗어난 가격 데이터

## 💡 해석 가이드

### 정규분포
- 종 모양의 대칭적 분포
- 가격이 평균 주변에 집중되어 있음
- 통계적 분석이 용이함

### 양의 왜도 (Right Skew)
- 오른쪽 꼬리가 긴 분포
- 높은 가격이 가끔 나타남
- 평균 > 중앙값

### 음의 왜도 (Left Skew)
- 왼쪽 꼬리가 긴 분포
- 낮은 가격이 가끔 나타남
- 평균 < 중앙값

### 이중봉 분포
- 두 개의 봉우리를 가진 분포
- 서로 다른 가격대의 데이터가 혼재
- 시장 상황의 변화를 의미할 수 있음

## 📊 주요 특징
- **히스토그램**: 막대 그래프로 빈도 표시
- **KDE 곡선**: 부드러운 곡선으로 분포 형태 표현
- **통계선**: 평균(빨간색), 중앙값(초록색) 표시
- **투명도**: 히스토그램에 투명도 적용으로 KDE 곡선과 겹침 방지

## 🎯 활용 방안
- **가격 안정성**: 분포의 집중도로 안정성 평가
- **이상치 탐지**: 비정상적인 가격 데이터 식별
- **통계적 모델링**: 분포 형태 기반 모델 선택
- **리스크 평가**: 분포의 산포도로 변동성 파악

## 📈 통계적 의미
- **좁은 분포**: 가격이 안정적이고 예측 가능
- **넓은 분포**: 가격이 불안정하고 변동성이 큼
- **대칭 분포**: 가격 상승과 하락이 균형적
- **비대칭 분포**: 특정 방향으로 편향된 가격 움직임
