# 🎨 Shared Assets

여러 프로젝트에서 공통으로 사용되는 이미지와 자산들이 저장된 폴더입니다.

## 📁 폴더 구조
- `images/character-images/` - 캐릭터 이미지 (man.png, woman.png)
- `images/Images/` - 일반 이미지 (img01.png ~ img10.jpg)

## 🖼️ 이미지 사용법
- **캐릭터 이미지**: 프로필 사진, 아바타 등에 사용
- **일반 이미지**: 피드 포스트, 갤러리 등에 사용

## 📝 사용 예시
```html
<!-- 캐릭터 이미지 사용 -->
<img src="../shared-assets/images/character-images/man.png" alt="Male Character">

<!-- 일반 이미지 사용 -->
<img src="../shared-assets/images/Images/img01.png" alt="Sample Image">
```
