# 📊 Yahoo Finance 주식 데이터 시각화 해석 가이드

이 가이드는 Yahoo Finance에서 수집한 주식 상승률 데이터의 시각화 차트들을 해석하는 방법을 설명합니다.

## 📁 생성된 시각화 파일들

다음 7개의 PNG 이미지 파일이 `visualizations/` 폴더에 생성되었습니다:

1. **price_change_distribution.png** - 가격 변동 분포
2. **top_performers_chart.png** - 상위 성과 주식
3. **pe_ratio_analysis.png** - PER 비율 분석
4. **volume_analysis.png** - 거래량 분석
5. **performance_categories.png** - 성과 구간별 분석
6. **correlation_heatmap.png** - 상관관계 히트맵
7. **summary_dashboard.png** - 종합 대시보드

---

## 📈 1. 가격 변동 분포 (Price Change Distribution)

### 파일: `price_change_distribution.png`

**차트 구성:**
- 왼쪽: 가격 변동(달러) 히스토그램
- 오른쪽: 변동률(%) 히스토그램

**해석 방법:**
- **분포 형태**: 정규분포에 가까운지, 왜도(skewness)가 있는지 확인
- **중심값**: 평균 가격 변동과 변동률 확인
- **산포도**: 데이터가 얼마나 퍼져있는지 확인 (표준편차)
- **이상치**: 극단적인 값들이 있는지 확인

**투자 인사이트:**
- 대부분의 주식이 어느 범위에서 변동하는지 파악
- 극단적인 상승을 보인 주식들 식별
- 시장 전체의 변동성 수준 평가

---

## 🏆 2. 상위 성과 주식 (Top Performers Chart)

### 파일: `top_performers_chart.png`

**차트 구성:**
- 상위 10개 주식의 변동률을 막대 차트로 표시
- 각 막대 위에 정확한 변동률 수치 표시

**해석 방법:**
- **순위**: 변동률 기준 상위 주식들 확인
- **격차**: 1위와 10위 사이의 성과 격차 확인
- **집중도**: 상위 주식들의 성과가 얼마나 집중되어 있는지 확인

**투자 인사이트:**
- 가장 주목할 만한 상승 주식들 식별
- 섹터별로 어떤 종목들이 상위에 있는지 분석
- 상승 모멘텀이 지속될 가능성 평가

---

## 📊 3. PER 비율 분석 (PE Ratio Analysis)

### 파일: `pe_ratio_analysis.png`

**차트 구성:**
- 왼쪽: PER 비율 분포 히스토그램
- 오른쪽: PER 비율 vs 변동률 산점도 (색상은 가격 변동 크기)

**해석 방법:**
- **PER 분포**: 대부분의 주식이 어느 PER 범위에 있는지 확인
- **상관관계**: PER과 변동률 사이의 관계 확인
- **색상 패턴**: 가격 변동이 큰 주식들의 PER 특성 확인

**투자 인사이트:**
- **저PER 주식**: 밸류 스톡으로 분류 가능
- **고PER 주식**: 성장주로 분류 가능
- **PER과 성과의 관계**: 밸류 vs 성장 전략의 효과성 평가

---

## 📈 4. 거래량 분석 (Volume Analysis)

### 파일: `volume_analysis.png`

**차트 구성:**
- 왼쪽: 거래량 분포 (로그 스케일)
- 오른쪽: 거래량 vs 변동률 산점도 (색상은 가격 변동 크기)

**해석 방법:**
- **거래량 분포**: 대부분의 거래량이 어느 범위에 있는지 확인
- **거래량과 성과**: 거래량이 많을수록 변동률이 높은지 확인
- **색상 패턴**: 가격 변동이 큰 주식들의 거래량 특성 확인

**투자 인사이트:**
- **거래량 급증**: 관심도가 높은 주식들 식별
- **유동성**: 거래하기 쉬운 주식들 파악
- **모멘텀**: 거래량과 가격 변동의 상관관계로 모멘텀 확인

## 🎯 5. 성과 구간별 분석 (Performance Categories)

### 파일: `performance_categories.png`

**차트 구성:**
- 왼쪽: 성과 구간별 분포 파이 차트
- 오른쪽: 구간별 평균 가격 변동 막대 차트

**성과 구간:**
- **고성과 (≥20%)**: 20% 이상 상승한 주식
- **중성과 (10-20%)**: 10-20% 상승한 주식
- **저성과 (<10%)**: 10% 미만 상승한 주식

**해석 방법:**
- **분포 비율**: 각 구간에 속한 주식의 비율 확인
- **평균 가격 변동**: 구간별 평균 가격 변동 크기 비교
- **균형성**: 시장이 균형잡힌 상승을 보이는지 확인

**투자 인사이트:**
- **시장 상황**: 전체적인 시장 상승세의 강도 평가
- **투자 전략**: 각 구간별로 다른 투자 전략 적용 가능
- **리스크 관리**: 고성과 구간의 비율로 시장 리스크 평가

## 🔗 6. 상관관계 히트맵 (Correlation Heatmap)

### 파일: `correlation_heatmap.png`

**차트 구성:**
- 각 지표 간의 상관계수를 색상으로 표시
- 빨간색: 양의 상관관계, 파란색: 음의 상관관계

**지표들:**
- **Price_Change**: 가격 변동 (달러)
- **Change_Percent_Numeric**: 변동률 (%)
- **PE_Ratio**: 주가수익비율
- **Volume_Numeric**: 거래량

**해석 방법:**
- **상관계수 범위**: -1 (완전 음의 상관) ~ +1 (완전 양의 상관)
- **0.7 이상**: 강한 양의 상관관계
- **-0.7 이하**: 강한 음의 상관관계
- **0.3~0.7**: 중간 정도의 상관관계

**투자 인사이트:**
- **지표 간 관계**: 어떤 지표들이 함께 움직이는지 파악
- **예측 가능성**: 한 지표로 다른 지표를 예측할 수 있는지 확인
- **다각화**: 상관관계가 낮은 지표들로 포트폴리오 다각화

## 📊 7. 종합 대시보드 (Summary Dashboard)

### 파일: `summary_dashboard.png`

**차트 구성:**
- 6개의 서브플롯으로 구성된 종합 대시보드
- 상위 5개 주식, 분포 차트들, 산점도, 파이 차트, 통계 요약

**해석 방법:**
- **한눈에 보기**: 모든 주요 지표를 한 번에 확인
- **패턴 인식**: 여러 차트에서 공통된 패턴 찾기
- **통계 요약**: 핵심 수치들을 빠르게 파악

**투자 인사이트:**
- **전체적인 시장 상황**: 시장의 전반적인 분위기 파악
- **투자 기회**: 가장 주목할 만한 주식들 식별
- **리스크 평가**: 시장의 변동성과 리스크 수준 평가

## 🎯 차트 해석을 위한 실용적 팁

### 1. 패턴 인식
- **트렌드**: 전체적인 방향성 확인
- **클러스터링**: 비슷한 특성을 가진 주식들 그룹화
- **이상치**: 일반적인 패턴에서 벗어난 주식들 주목

### 2. 상관관계 활용
- **양의 상관**: 함께 움직이는 지표들
- **음의 상관**: 반대로 움직이는 지표들
- **무상관**: 독립적으로 움직이는 지표들

### 3. 통계적 유의성
- **표본 크기**: 25개 주식으로 제한적이므로 일반화에 주의
- **시점**: 특정 시점의 데이터이므로 시장 상황 고려
- **섹터 편향**: 특정 섹터에 편중될 수 있음

### 4. 투자 결정에 활용
- **스크리닝**: 차트를 통해 관심 주식들 선별
- **리스크 관리**: 변동성과 거래량으로 리스크 평가
- **진입/청산 시점**: 모멘텀과 밸류에이션으로 타이밍 결정

---

## ⚠️ 주의사항

1. **과거 데이터**: 이 차트들은 과거 데이터를 기반으로 하므로 미래 성과를 보장하지 않습니다.

2. **시장 상황**: 특정 시점의 데이터이므로 시장 상황에 따라 결과가 달라질 수 있습니다.

3. **투자 조언 아님**: 이 분석은 참고용이며, 투자 결정은 신중하게 하시기 바랍니다.

4. **추가 분석**: 더 정확한 분석을 위해서는 더 많은 데이터와 시간대가 필요합니다.
